Web Application

Budget Cloud

     Budget Cloud is a web application that provides cloud computing architecture estimates from 4 major cloud providers: GCP, AWS, Azure, and Oracle. It uses daily refreshed live data and a user form to calculate estimates based on the user's input. The frontend is built using React.js, while the backend is powered by Django with SQLite for data storage.

Crypto Risk Analyzer

     Crypto Risk Analyzer is a simple tool that shows real-time crypto risk scores by pulling live market data from a public API and tracking signals like volatility and liquidity. The frontend is built using React.js, while the backend is powered by Django with SQLite for data storage.
